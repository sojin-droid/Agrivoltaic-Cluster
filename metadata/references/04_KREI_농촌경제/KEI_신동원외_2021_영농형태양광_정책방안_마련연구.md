# 영농형 태양광 추진을 위한 정책방안 마련 연구

**출처**: 한국환경연구원 (구 환경정책평가연구원, KEI) 신동원 외 (2021)
**KISTI URL**: https://scienceon.kisti.re.kr/srch/selectPORSrchReport.do?cn=TRKO202200017649
**보고서번호**: TRKO202200017649

## 본 프로젝트 활용 핵심

이 보고서는 **한국 영농형 태양광 정책 권고의 원천(progenitor)** 문서로,
이후의 NARS 2024-02, KEEI 2024-14호, 농식품부 2024.4 도입전략이 모두 이를 인용합니다.

### 핵심 정책 권고 (보고서 5장)

**[발전사업 측면]**
1. **REC 가중치 상향** (현행 RPS 하에서 초기부담 완화) ← 본 문서 PPA 분석과 결합 가능
2. **FIT → FIP 전환 검토** (시장가격 연동 + 보조금)
3. **장기저리융자 20년 이상 + 거치기간 유연성**
4. **집적화된 사업 수행 지역 선정·운영** ← **본 프로젝트 "클러스터" 컨셉의 정책 권고 원천**
5. **마을·농민참여형 사업 인센티브** + 조합 선별 지원

**[농지·법제 측면]**
6. 자경농지·소규모 농지·진흥구역 외 등 **현행법 내 추진 가능 지역 우선**
7. 농지법 시행령 개정으로 **일시사용허가 20~25년 연장** ← 2024년 농식품부 23년 발표의 원안
8. 영농형 태양광 발전설비를 **"농업생산보조시설"로 규정** + 세제 지원 + 직불금 유지
9. 식량안보 우려 적은 품목 선별 (녹차 등) — 비적합 품목은 금융지원 제외

**[수용성 측면]**
10. **"영농형 태양광 도입을 위한 계획서" 제출 의무화** (일본 사례)
11. 지역주민 협의 사전확인 의무
12. **임차농 소득 감소분 보장** — "주민조합형 영농 태양광 모델"

## 본 프로젝트 시사점

### Bardach 7단계 (Decide) — 정책 도구 풀(pool)

본 보고서가 제시한 12개 정책 도구는 본 문서의 **"정책 옵션 메뉴"**로 직접 사용 가능합니다.
본 문서 권고 표를 다음처럼 재구성:

```
후보 클러스터 → 적합한 정책 도구 조합 매핑
- 평택 대추리 (국공유): #1(REC) + #4(집적화) + #6(현행법 내) + #10(계획서)
- 당진 송산 (법인 농업보호구역): #4 + #7(허가기간 연장) + #11(주민협의) + #12(임차농 보장)
```

### 데이터 한계 보강

본 보고서가 인용한 **한국 영농형 태양광 잠재량 682GW (농지 15,760km²)** 추정치가
KEEI 2024-14호로 확산된 출처 — 본 프로젝트 후보지(평택 381MW + 당진 654MW + 서산 등)의
전국 잠재량 대비 비율을 계산할 때 기준값으로 활용.

본 프로젝트 합산: ~1,700MW (4개 후보) = 전국 잠재량의 0.25%
→ "13개 시군 산단 10km 반경 내 후보만으로도 ~1.7GW" 메시지 가능
