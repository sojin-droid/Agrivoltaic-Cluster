## 학습 포인트 — "정책 멀티플레이어" 체크리스트

각 작업이 끝날 때 자기 진단:

- [ ] 권고가 **누구를 향한 것인가** 명시했는가?
- [ ] 권고에 **반대할 사람이 누구이고 왜인가** 답했는가?
- [ ] 결과 추정에 **불확실성 범위**를 명시했는가?
- [ ] **trade-off**를 정직하게 드러냈는가, 아니면 좋은 면만 강조했는가?
- [ ] 의사결정자가 **"그래서 내일 무엇을 하라"**고 묻는다면 답이 있는가?
- [ ] 데이터·근거의 **출처를 추적 가능**하게 인용했는가?
- [ ] 분석가의 **개인 견해**와 **데이터 근거**가 구분되는가?

이 7개 체크가 모두 ✅이면 정책 분석가 자격이 있다고 봐도 됩니다.
